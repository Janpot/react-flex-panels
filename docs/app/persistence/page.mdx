# Persistent Panel Positions

Panel positions are automatically persisted to localStorage when you provide `persistenceId` props. This enables panel states to survive page refreshes and sync across browser tabs.

## How it works

- Add `persistenceId` props to panels that should persist their state
- Panel positions are automatically saved to localStorage
- Changes sync across tabs and browser windows in real-time
- Panel IDs must be unique across your application
- Non-persistent panels use generated IDs that reset on unmount
