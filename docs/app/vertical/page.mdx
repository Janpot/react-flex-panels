# Vertical Layout

Create vertical layouts using the direction prop. Perfect for header/content/footer layouts or any vertically stacked interface.

## Demo

<DemoSection
  demo={<VerticalDemo />}
  tsSource={verticalSrc}
  cssSource={verticalCss}
/>

## How it works

- The `direction="column"` prop creates a vertical flex container
- The header has a fixed height of `60px` (fixed size)
- The content area has no size prop, so it automatically fills remaining space
- The footer has a fixed height of `40px` (fixed size)
